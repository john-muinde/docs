---
title: "Name Lookup"
openapi: "POST /api/v3/name-lookup"
---

Validate the mobile money account name for a phone number before sending a payout to it, reducing the risk of sending funds to the wrong recipient. Pass the phone number, network, and country directly — no prior customer record is required. This endpoint is only available to integrators approved for name-lookup. Currently supported for corridors handled by the assigned provider; unsupported corridors return `status: "UNSUPPORTED"` rather than an error.
